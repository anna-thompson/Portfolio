Projects

Electrical Engineering Calculator

During my internship at Sierra Nevada Corporation (SNC), I developed an application that performs a variety of electrical engineering calculations using C# and the .NET framework. This tool was designed to solve complex calculations commonly used by electrical engineers at SNC, making these calculations much more efficient to perform and do all in one place. The application featured a user-friendly interface with dropdowns, input fields, and validation to ensure accurate data entry. My role included not only coding the calculations but also designing the UI, implementing data validation, and ensuring the tool’s reliability by writing unit tests for each calculation.

Online Chess Platform

As part of a team project for my Object-Oriented Design course at Colorado State University (CSU), we are developing an online chess platform where players can create unique profiles, challenge opponents, and engage in turn-based, asynchronous matches. This project is mostly focused on designing our system based on established design patterns and strategies to produce a maintainable and scalable system. This project involved the use of Java for backend development, and JavaScript and React for our frontend.

Trip Planner

As part of a team project for my Software Engineering course at CSU, we designed and developed a trip planner application focused on optimizing travel routes. The application enabled users to select multiple destinations on a map and, using an algorithm, calculated the most efficient route to visit each location. The application also enabled users to search for specific airports by name or find nearby airports based on provided coordinates, using a comprehensive database of known airports.

Food Frenzy

As part of my Exploring Virtual Worlds course, I designed and developed a fast-paced cooking game built in Unity, inspired by the game Overcooked! The gameplay focuses on speed, coordination, and precision, requiring players to chop ingredients, cook dishes, and deliver orders accurately to meet the demands of customers.

Raspberry Pi Home Security System

At CSU in my Operating Systems course, I led a team to develop a home security system using a Raspberry Pi. We designed a program using Python that uses a camera module to detect and log faces when they enter the frame. To achieve this, we implemented the OpenCV library, using its computer vision and machine learning capabilities to identify faces. This made our system efficient for real-time security monitoring.
